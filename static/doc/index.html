Theme Layout

Liquid Layout - Full

The most relaxed method of providing a dynamically expanding or contracting design makes use of the ever-popular percentage (%) unit measurement. This layout type has gained mass popularity because it is the ultimate way of allowing the total opposite of a fixed layout where the content will simply take whatever space is available to it.


Fluid Layout - Boxed


NOTE - add boxed class to admindo theme wrapper element to enable/use the boxed fluid layout. See below code.

<div id="wrapper" class="boxed">
...
</div>

NOTE - to enable background pattern/image for the boxed fluid layout, add background pattern/image class to the page body element. See below code.

<body class="body1">

CSS File Structure - top

Below code shows the example of required CSS file that use in this themes. The CSS file must be included in the sequence below.

Required CSS File can be found in css folder.

  1. MASTER CSS
    • jasny-bootstrap.css
    • bootstrap.css
  2. ICONS CSS
    • fugue.css
    • elusive.css
    • font-awesome.css
  3. SITE CSS
    • default.css
    • default.responsive.css
  4. PLUGIN CSS
    • Important! the plugin CSS file is require if you're using javascript plugin. Example if an image gallery plugin is use, you need to include the plugin CSS file. See Plugin Dependencies for more detail.
  5. JQUERY UI CSS
    • jquery.ui.css

TIPS include the minify version of CSS file(filename.min.css) to reduce load time.

<!-- ===================== MASTER CSS ===================== -->
<link rel="stylesheet" type="text/css" href="css/jasny-bootstrap.min.css">
<link rel="stylesheet" type="text/css" href="css/bootstrap.min.css">
  
<!-- ===================== ICONS CSS ===================== -->
<link rel="stylesheet" type="text/css" href="css/icon/fugue.min.css">
<link rel="stylesheet" type="text/css" href="css/icon/elusive.min.css">
<link rel="stylesheet" type="text/css" href="css/icon/font-awesome.min.css">
  
<!-- ===================== SITE CSS ===================== -->
<link rel="stylesheet" type="text/css" href="css/default.min.css">
<link rel="stylesheet" type="text/css" href="css/default.responsive.min.css">
  
<!-- ===================== PLUGINS CSS ===================== -->
  
<!-- ===================== JQUERY UI CSS ===================== -->
<link rel="stylesheet" type="text/css" href="css/jui/jquery.ui.min.css">

Javascript File Structure - top

The javascript file is located in js folder. The are two type of javascript file, minify(filename.min.js) and development(filename.js). The development file is located in src folder.

Note that the below javascript file is require in this theme. Include the javascript file at bottom of the page. If you using for an example calendar plugin, you must include the plugin file(javascript) at the plugin section. See the below code example.

TIPS include the minify version of Javascript file(filename.min.js) to reduce load time.

<!-- Vendors -->
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/jqueryui/1.9.2/jquery-ui.min.js"></script>
  
<!-- Site -->
<script src="js/bootstrap.min.js"></script>
<script src="js/jasny-bootstrap.min.js"></script>
<script src="js/application.js"></script>
  
<!-- Plugin -->
